## Sobre El Mirador de Morelos

El Mirador de Morelos busca ser un portal web que sirva como guía turística al proporcionar principalmente un catálogo de lugares de interés en cada municipio del estado. Para esta aplicación se pretenden cumplir los siguientes puntos.
- Contar con un buscador que permita filtrar lugares/sitios/negocios por categoría: ej. gym, restaurante, eventos culturales.
- Debe contar contar con una sección de publicaciones donde el usuario podrá visualizar tanto artículos, noticias o anuncios publicitarios.
- Cada publicación debe poder ser interactuable para poder proporcionarle al usuario los medios de contacto respectivos.

## Reglas de negocio

1.  El sistema debe ser un portal web que facilite la búsqueda de negocios, servicios y actividades de los diferentes municipios dentro del estado de Morelos. 
2.  Los negocios, servicios y actividades deben estar organizados mediante categorías para facilitar la búsqueda.
3.  Todo negocio o servicio registrado deberá incluir, nombre, descripción, categoría, municipio, ubicación, redes sociales, medios de contacto. 
4.  El portal debe incluir una sección de artículos/noticias que deben poder filtrarse por fecha, categoría y municipio. 
Al hacer clic en un artículo o noticia, se deben poder visualizar las redes sociales asociadas a ese contenido. 
5.  El sistema debe manejar dos tipos principales de usuarios:
      - Dueños o representantes de los negocios o servicios. 
      - Usuarios visitantes.
